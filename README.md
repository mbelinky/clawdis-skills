# Clawdis Skills

Custom skills for Clawdis/Razor built by Mariano Belinky.

## Skills

### [twitter-bookmarks](./twitter-bookmarks)
Automated Twitter bookmark processor with YAML-configurable categories, LLM analysis, and smart routing.

- **Language:** Go
- **Status:** Production (running every 20 min via cron)
- **Features:** Gemini categorization, content summarization, Obsidian integration, Telegram notifications
